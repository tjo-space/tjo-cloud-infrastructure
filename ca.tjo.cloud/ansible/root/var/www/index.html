Configuring Linux to accept CA

        sudo mkdir -p /usr/local/share/ca-certificates/ca-tjo-cloud
        sudo curl -sLo /usr/local/share/ca-certificates/ca-tjo-cloud/root.crt https://ca.tjo.cloud/root.crt
        sudo update-ca-certificates
